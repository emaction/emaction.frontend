# Easy to use GitHub Styled emoji reaction for Blogs.

## Get Started

### Via CDN (Recommended)
`<script type="module" src="https://cdn.jsdelivr.net/gh/emaction/dist@1.0.0/bundle.min.js"></script>`

### NPM
`npm install emaction`
